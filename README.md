# XTrack


# FrontEnd
* cd frontend
* npm install
* npm start

# Backend
* cd api
* npm install 
* npm start

# Deploy to versel
* In frontend  folder run npm build
* copy build folder contents  to  root public folder

* add vercel.json

* copy all the backend to api folder 

* https://vercel.com/new   => Continue with github, Add the repo , deploy with default settings.
